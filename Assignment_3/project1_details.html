Client Website

I successfully delivered my first client project using React, creating a fully functional website with responsive design for various screens.

The website is designed for a Construction Company, based in Johannesburg and includes features such as navigating through different pages of the website.

As the sole front-end developer, I implemented navigation through different pages using REACT Router library and ensuring a seamless user experience across devices. I also made sure that there is consistency with the colors of the website.

Technologies used: React and CSS for styling.

Challenges: The challenge That I faced during the project, was working with images on the cards, The size of the images kept on changing across the cards.

TimeLine:It took me 2 weeks to complete the project.
